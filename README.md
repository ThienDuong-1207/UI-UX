## Overview
This repository contains UI/UX practice assignments from Lab 1 to Lab 6, including static web projects (HTML/CSS), a React project, and multi-screen interface design exercises.

## Lab Descriptions

### Lab 1 - Dak Lak Coffee (HTML/CSS)
- Type: Static website (without JavaScript)
- Folder: `lab1/`
- Main content:
  - Build homepage, products page, and about page
  - Apply responsive layout with CSS Grid/Flexbox
  - Use hover effects, animations, and CSS variables

### Lab 2 - Basic HTML/CSS
- Type: Foundational practice
- Folder: `lab2/`
- Main content:
  - Implement a basic interface in `index.html`
  - Practice layout, components, and simple website structure

### Lab 3 - Ohana (React + Vite)
- Type: React website
- Folder: `lab3/`
- Main content:
  - Build a room search/rental website
  - Split UI into components (Header, Hero, RoomList, RoomDetail, Contact, Footer)
  - Use routing for navigation and room detail pages

### Lab 4 - Product/Homescreen UI
- Type: Product interface design exercise
- Folder: `lab4/`
- Main content:
  - Design home screen and product detail screen
  - Follow the Precision Engineering style direction (modern, minimalist, spec-focused)
  - Standardize colors, typography, and spacing based on design documents

### Lab 5 - Food Delivery UI (Mobile)
- Type: Food delivery mobile UI system
- Folder: `lab5/`
- Main content:
  - Implement multiple screens: splash, sign in, home, restaurant detail
  - Build the Vibrant Cravings style guide (appetizing colors, soft corners, strong CTAs)
  - Keep a consistent visual language across screens

### Lab 6 - Food Delivery UI (Web)
- Type: Web version of the food delivery project
- Folder: `lab6/`
- Main content:
  - Convert/adapt key screens into web layouts
  - Maintain consistency in color, typography, and components with the style guide
  - Improve visual hierarchy and user experience for large screens
